# Колеса Свободы - Сервис аренды автомобилей

Веб-приложение для компании автопарка "Колеса Свободы", разработанное на Flask.

## Описание проекта

Сайт предоставляет возможность бронирования автомобилей с разграничением прав доступа для пользователей и администраторов.

### Основной функционал:
- Регистрация и авторизация пользователей
- Просмотр каталога автомобилей
- Бронирование автомобилей
- Административная панель для управления заявками

### Технологии:
- Python Flask
- MySQL
- HTML/CSS/JavaScript

## Структура проекта:
```
├── routes/
│   ├── admin.py         # Маршруты административной панели
│   ├── auth.py          # Аутентификация и авторизация
│   ├── create_db.py     # Создание базы данных
│   ├── init_db.py       # Инициализация базы данных
│   └── users.py         # Маршруты пользователей
├── static/
│   ├── css/            # Стили
│   └── js/             # JavaScript файлы
├── templates/          # HTML шаблоны
└── main.py            # Точка входа приложения
```

## База данных
Структура включает следующие таблицы:
- clients (клиенты)
- trips (поездки)
- cars (автомобили)
- bookings (бронирования)

## Установка и запуск

1. Клонировать репозиторий
2. Создать и активировать виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # для Unix
venv\Scripts\activate     # для Windows
```
3. Установить зависимости:
```bash
pip install -r requirements.txt
```
4. Создать и инициализировать базу данных:
```bash
python routes/create_db.py
python routes/init_db.py
```
5. Запустить приложение:
```bash
python main.py
```
